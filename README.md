## Reactive programming - Mono and Flux Playground
Flux and Mono hands-on project which covers the following functions along with test cases.

* Mono.just()
* Flux.fromIterable()
* Maps with Flux
* Lists with Flux
* filter()
* flatMap()
* concatMap()
* flatMapMany()
* Functional interfaces with transform()
* defaultIfEmpty()
* switchIfEmpty()
* concat()
* concatWith()
* merge()
* mergeWith()
* mergeSequential()
* zip()
* zipWith()
* fromArray()
* delayElements()
* logs()
